# 01. 요구사항 정의 프롬프트

아래 내용을 채운 뒤 Codex에 그대로 붙여넣어 사용하세요.

```text
나는 AI 서비스 개발 프로젝트를 시작하려고 해.
아래 아이디어를 바탕으로 AI 서비스 개발용 요구사항 정의서를 작성해줘.

## 프로젝트 아이디어
- 만들고 싶은 서비스:
- 해결하고 싶은 문제:
- 예상 사용자:
- 사용자가 입력할 데이터:
- 서비스가 제공할 결과:

## 개발 조건
- 수업 또는 제출 기한:
- 팀원 수:
- 사용하고 싶은 기술 스택:
- 반드시 포함하고 싶은 기능:
- 제외해도 되는 기능:

## 요청 사항
다음 항목을 포함해서 Markdown 형식의 요구사항 정의서를 작성해줘.

- 프로젝트 개요
- 해결하고자 하는 문제
- 대상 사용자
- 입력 데이터
- 핵심 기능
- 처리 흐름
- 출력 결과
- MVP 범위
- 추가 기능
- 확인 필요 사항

아직 정보가 부족한 부분은 임의로 확정하지 말고 "가정 사항" 또는 "확인 필요 사항"으로 표시해줘.
```
